Motivation:

Efficient Storage: Number plate images captured at traffic signals need to be stored for various purposes, such as traffic monitoring, law enforcement, or investigations. Since a large number of number plate images are captured every day, efficient storage becomes crucial. SVD-based compression techniques can significantly reduce the storage requirements by representing the image using fewer singular values, without significant loss in visual quality.


Technical terms:

  1. Matrix: A matrix is a two-dimensional array of numbers arranged in rows and columns. In the context of image compression using SVD, an image can be represented as a matrix where each element corresponds to a pixel value.

  2. Singular values are the diagonal elements of the matrix obtained during the SVD decomposition of a matrix. For a given matrix A, the singular value decomposition can be represented as A = UΣV^T, where U and V are orthogonal matrices, and Σ is a diagonal matrix containing the singular values. Singular values are non-negative and sorted in descending order on the diagonal of the Σ matrix. The first singular value is the largest, followed by the second, and so on. The magnitude of the singular values indicates the relative importance of each singular vector in capturing the information present in the matrix.

Singular Value Decomposition:

SVD decomposes a matrix A into three matrices: U, Σ, and V^T. Mathematically, A = UΣV^T, where U and V are orthogonal matrices, and Σ is a diagonal matrix containing the singular values. In Matlab we can perform Singular Value Decomposition using 'svd' function. Here's an example of MATLAB code for performing Singular Value Decomposition (SVD) on a matrix:

% Example matrix
A = [1, 2, 3; 4, 5, 6; 7, 8, 9];
% SVD decomposition
[U, S, V] = svd(A);
% U: Left singular vectors
disp("Left singular vectors (U):");
disp(U);
% S: Singular values
disp("Singular values (S):");
disp(S);
% V: Right singular vectors
disp("Right singular vectors (V):");
disp(V);

Algorithm to finding SVD:

    Let A in Mn×d,
  1. Find AA^T or A^T A.
  2. Decompose AAT or AT A by Eigen-value decomposition,
    i.e. find characteristic value and normalized characteristic vector of AAT or AT A.
  3. Construct Σ by using square root of characteristic value of AAT or AT A.
  4. Construct U or V T by characteristic vector of AAT or AT A respectively.
  5. Use A = UΣV T to obtain missing matrix.
