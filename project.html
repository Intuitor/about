Mahindra University (June 2023- Aug 2023):

Abstract: In this project, We studied the theory of Group Actions, proof of Sylow theorems via Group Actions and theory of nilpotent groups. This report omits an in-depth presentation of commutators, nilpotent groups and proofs of Sylow theorems but it does present basic properties and lemmas that are essential for further exploration of prime power groups and to study the structure of subgroups generated by small conjugacy classes in finite groups. We explored the relationship between the ’number of different sizes of conjugacy classes and the underlying ”Group structure.” Specifically, We studied how the con- jugacy class sizes in a group can provide insights into its properties. For instance, if a group has only conjugacy classes of size one, it must be an Abelian group. Our main aim was to study the structure of groups with having only two different size of conjugacy classes. In particular we concentrated on results given by Avinoam Mann.

Technical terms:

  1. Matrix: A matrix is a two-dimensional array of numbers arranged in rows and columns. In the context of image compression using SVD, an image can be represented as a matrix where each element corresponds to a pixel value.

  2. Singular values are the diagonal elements of the matrix obtained during the SVD decomposition of a matrix. For a given matrix A, the singular value decomposition can be represented as A = UΣVT, where U and V are orthogonal matrices, and Σ is a diagonal matrix containing the singular values. Singular values are non-negative and sorted in descending order on the diagonal of the Σ matrix. The first singular value is the largest, followed by the second, and so on. The magnitude of the singular values indicates the relative importance of each singular vector in capturing the information present in the matrix.

Singular Value Decomposition:

SVD decomposes a matrix A into three matrices: U, Σ, and VT. Mathematically, A = UΣVT, where U and V are orthogonal matrices, and Σ is a diagonal matrix containing the singular values. In Matlab we can perform Singular Value Decomposition using 'svd' function. Here's an example of MATLAB code for performing Singular Value Decomposition (SVD) on a matrix:

% Example matrix
A = [1, 2, 3; 4, 5, 6; 7, 8, 9];
% SVD decomposition
[U, S, V] = svd(A);
% U: Left singular vectors
disp("Left singular vectors (U):");
disp(U);
% S: Singular values
disp("Singular values (S):");
disp(S);
% V: Right singular vectors
disp("Right singular vectors (V):");
disp(V);

Algorithm to finding SVD:

    Let A in Mn×d,
  1. Find AAT or AT A.
  2. Decompose AAT or AT A by Eigen-value decomposition,
    i.e. find characteristic value and normalized characteristic vector of AAT or AT A.
  3. Construct Σ by using square root of characteristic value of AAT or AT A.
  4. Construct U or VT by characteristic vector of AAT or AT A respectively.
  5. Use A = UΣVT to obtain missing matrix.
